## 正文

🌱 起点其实很私人

一年前我开始遇到一个老问题：
想做的事散在备忘录里，
钱记在另一个 App 里，
读过的东西、和朋友的互动、自己的反思，又在另外三个地方。
我换过 Notion、Things、MoneyWiz、Obsidian，但每个都只解决一格。

于是我做了 **NexaLife（构筑人生）**。
最开始它只是我自己的工作台——只为了让"想到 → 收进来 → 分流 → 执行 → 复盘 → 沉淀"这条路，**只发生在一个 App 里**。

---

🛠️ 它是什么

一句话：**macOS 上的本地优先人生操作系统**。

六个模块，覆盖一个普通人会反复回到的事：
- 📥 **收件箱 Inbox** — 想到什么先按 ⌘⌥N 丢进来
- ✅ **执行 Execution** — 看板 + 项目，长期 / 中期 / 短期标得清清楚楚
- 💰 **生活 Lifestyle** — 账务、目标、人脉，AI 帮你分类 & 建议互动策略
- 📚 **知识 Knowledge** — 按主题写下来，AI 可以一键生成摘要
- 🧘 **觉知 Vitals** — 情绪、反思、核心守则
- 📊 **仪表盘 Dashboard** — 一眼看到这个月你在往哪走

数据**默认全部本地存储**，AI 是可选加速器——没配 API Key 也能完整用。

---

✨ v0.2.0 做了什么

这一版的主线是把 **AI 真正织进每个模块**，而不是当成一个开关：
- Quick Capture 写一句话，AI 实时建议归到哪个模块
- 记一笔账，AI 自动读标题和金额，给你建议分类
- 新建任务，AI 自动填好分类、标签和所属项目
- 每个联系人，AI 给出重要性评分 + 下一步互动建议
- 仪表盘上有一个**始终可见的 AI Mentor**，随时可以打开对话

外加：执行模块项目行重做、Cherry Studio 风格的模型选择器、Onboarding 加了 AI Mentor 配置步骤。

---

👥 我希望它适合谁

我想了很久这个版本想给谁用，最后写下来其实就两类人：

**1️⃣ 已经习惯给自己做规划，想"看见"自己每一步的人**
你不需要我教你怎么用——你只是缺一个能把目标、任务、复盘、账务、人脉**串起来**的工作台。

**2️⃣ 还没有规划习惯，但想培养，并希望 AI 给出建议的人**
你只要正常用，AI 会基于你过去的记录主动告诉你：
"你最近的支出在向哪类倾斜""这个月你写得最多的反思关键词是什么""下一步可以做什么"。
不是替你决定，而是把你看不到的模式摆到你面前。

我没把它当一个完成品发布。说实话，这是我自己的工作流先固化下来，再"长大"成给别人也能用的版本。
