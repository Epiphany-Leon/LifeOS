## 文案 · 版本 A（推荐 · 轻松版）

NexaLife v0.2.0 出来啦 🌱

上次发 v0.1.1 还在说"刚把骨架搭起来"，这次终于把 AI 真的织进每个模块了——

我准备拉个**测试群**，欢迎来折腾、提需求、报 Bug——

## 文案 · 版本 C（情感叙事版）

从 v0.1.1 到 v0.2.0，差不多三个月。

v0.1.1 的时候它还只是"我自己的工作台"，
这一版我开始认真想：怎么让没有这个习惯的人也能用进来。

现在 AI 会帮你分模块、建议账务分类、给人脉提建议、
在仪表盘上看着你最近的记录给反馈——它不替你做决定，但会把你忽略的模式摆到你面前。

想找一些朋友先用起来，攒一波真实反馈再往下走。
建了个群，扣 1 拉你 🌱
